# Site Nina Carducci — Optimisation SEO

Projet 8 de la formation **Intégrateur Web** (OpenClassrooms).

## 🎯 Description

Optimisation du référencement et de l'accessibilité du site de Nina Carducci,
photographe basée à Bordeaux, dans le cadre d'une mission freelance.
Le projet consiste à auditer, corriger et améliorer les performances,
le SEO technique, le référencement local et l'accessibilité du site existant.

## ✨ Optimisations réalisées

- Compression et conversion des images en WebP
- Mise en place du lazy loading sur les images
- Amélioration de la structure sémantique HTML
- Optimisation des balises meta (title, description)
- Référencement local via microdonnées Schema.org
- Ajout des balises Open Graph et Twitter Cards
- Corrections d'accessibilité (contrastes, attributs alt, labels)

## 🛠️ Stack technique

- HTML5
- CSS3
- JavaScript
- Bootstrap
